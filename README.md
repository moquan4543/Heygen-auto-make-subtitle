# Heygen-auto-make-subtitle  
* * *
## **autosubtitle.py**  
流程:  
1.剪映輸出字幕選txt檔  
2.可以利用馴碼快手轉成繁體中文(可選)  
3.手動根據老師講的內容分好哪幾句話是同一頁PPT中講的，不同頁的內容用連續10至50個字母'a'來分隔  
4.將改好的txt檔讀入這隻程式裡(在main中指定好路徑)  
5.手動將進度條拉至第一張投影片，注意需要有add speech字樣  
6.執行程式  
## **convertSRT.py**  
可以利用此程式將srt檔轉換為純文字
